

# Projekt 1 - Budowanie niezależnych komponentów

Zacznijcie projekt od stworzenia aplikacji za pomocą komendy: npx create-react-app my-app.dev, lub vite
Stwórzcie nowego brancha, nastpępnie postawcie czysty projekt z reacta.
Pamiętajcie, zeby na koniec zajęć wypchnąć wszystko na brancha
Link pomocniczy:
https://create-react-app.dev
## Zadanie 1:
- utwórz nowy projekt aplikacji React
- utwórz niezależne komponenty:
  - `Header` - wyświetlający nazwę strony
  - `Content` - zawierający content aplikacji
  - `Footer` - wyświetlający napis: `Copyright 2022. Wszystkie prawa zastrzeżone.` - data powinna być dzisiejszym rokiem.
- wszystkie dane komponentów, powinny być przekazane dynamicznie i wyświetlone na podstawie mechanizmu `props`.
